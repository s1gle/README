Тег address

Тег address предназначен для хранения информации об авторе и может включать в себя любые элементы HTML вроде ссылок, текста, выделений и т.д. Планируется, что поисковые системы будут анализировать содержимое этого тега для сбора информации об авторах сайтов. Не следует использовать для оформления адресов электронной почты.

По умолчанию текст внутри тега address отображается курсивом. Это можно изменить с помощью CSS свойства font-style.
