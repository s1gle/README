Атрибут alt

Атрибут alt альтернативный текст, который будет показан, если браузер не может найти изображение, указанное в атрибуте src тега img. Является обязательным атрибутом для тега img, при его отсутствии будет ругаться валидатор (программа, которая проверяет корректность HTML или CSS). 


<img src="" alt="Картинка веселой обезьянки">
