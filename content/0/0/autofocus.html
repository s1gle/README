Атрибут autofocus

Атрибут autofocus задает автоматический фокус ввода по умолчанию для полей HTML формы при загрузке страницы.

Тот элемент, которому задан атрибут autofocus, будет иметь фокус при загрузке страницы (в нем будет моргать курсор, если начать вводить текст с клавиатуры - он будет попадать в поле с фокусом ввода).

Атрибут является атрибутом без значения. Его следует применять к тегам input или textarea и только одному элементу на странице, в противном случае будет конфликт.

псевдокласс focus, который задает стили элементу в фокусе ввода
атрибут tabindex, который позволяет менять фокус по нажатию Tab
