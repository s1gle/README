Псевдокласс active

Псевдокласс active задает стили для ссылки в момент нажатия на нее (нажали и не отпустили кнопку - пока мы ее не отпускаем, заданные стили работают).

Обычно этот псевдокласс применяется совместно с псевдоклассами link, hover и active.

Синтаксис 

a:active {
            
    }

Пример
Непосещенная ссылка будет красного цвета, посещенная - голубого. Если вы наведете на нее мышкой - она станет зеленого цвета, а если нажмете левую кнопку мыши и будете ее удерживать - станет черной (состояние active):

HTML
<a href="#">ссылка</a>
    a:link {
        color: red;
        }
    a:visited {
        color: blue;
        }
    a:hover {
        color: green;
        }
    a:active {
        color: black;
    }
