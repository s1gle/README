Методы hide и show в jQuery

Эффекты позволяют анимировать элемент, например - плавно скрыть или показать его. Для каждого эффекта существуют свои методы jQuery. Давайте начнем с методов hide и show. Методы hide и show позволяют скрывать и показывать элементы. Рассмотрим их работу.
Можно задавать время выполнения эффекта в миллисекундах (1000 миллисекунд = 1 секунда). В этом случае эффект будет плавно проигрываться в течении заданного времени. Давайте зададим время выполнения эффекта 1 секунду и посмотрим, как поменяется эффект: 

        $('#hide').click(function() {
            $('#elem').hide(1000);
        });
        
        $('#show').click(function() {
            $('#elem').show(1000);
        });

Методы slideUp и slideDown в jQuery

Методы slideUp и slideDown позволяют сделать "закатывание" и "выкатывание" элемента обратно (параметром они также принимают время исполнения эффекта, если его не указать - возьмется время по умолчанию, 400мс). Как это выглядит - смотрите на следующем примере. 

        $('#slipup').click(function() {
            $('#elem1').slideUp(400);
        });
        
        $('#slipdown').click(function() {
            $('#elem1').slideDown(400);
        });

Метод slideToggle в jQuery

Следующий метод - slideToggle работает так: если элемент скрыт - показывает его, а если показан - скрывает. Удобство этого метода в том, что это все можно повесить на одну кнопку, а не на две, как это было в предыдущих уроках. 

        $('#toggle').click(function() {
            $('#elem').slideToggle(1000);
        });

Методы fadeOut и fadeIn в jQuery

Методы fadeOut и fadeIn реализуют плавное исчезновение и появление элемента. 

        $('#fadeout').click(function() {
            $('#elem3').fadeOut(1500);
        });
        
        $('#fadein').click(function() {
            $('#elem3').fadeIn(1500);
        });

Метод fadeToggle в jQuery

Метод fadeToggle работает аналогично методу slideToggle, он тоже реализует чередование эффектов: то показывает элемент, то скрывает. Например, мы можем, нажимая на одну и ту же кнопку и показать и скрыть элемент. 

        $('#toggle1').click(function() {
            $('#elem4').fadeToggle(2000);
        });
